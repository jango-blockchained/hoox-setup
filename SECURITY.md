# Security Policy

## Supported Versions

Currently, only the latest version of the `main` branch is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| others  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Hoox, please send an e-mail to the maintainers instead of opening a public issue. We take security issues seriously and will respond to reports as quickly as possible.

- DO NOT create a public issue for security vulnerabilities.
- Provide a detailed description of the issue and steps to reproduce.
- Please allow us reasonable time to respond before making the vulnerability public.

We appreciate your efforts to responsibly disclose your findings, and will make every effort to acknowledge your contributions.
