# Wiki Sync Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Automatically sync the documentation to the `hoox-setup.wiki` GitHub repository upon pushes to the main branch.

**Architecture:** We will create a GitHub Actions workflow (`.github/workflows/wiki-sync.yml`) that clones the wiki repository, copies the contents of the `docs/` folder along with key markdown files (`README.md`, `AGENTS.md`, `CLI_FEATURES.md`), commits the changes, and pushes them to the wiki.

**Tech Stack:** GitHub Actions, Bash, rsync.

---

### Task 1: Create GitHub Actions Workflow for Wiki Sync

**Files:**

- Create: `.github/workflows/wiki-sync.yml`

- [ ] **Step 1: Write the GitHub Actions workflow file**

```yaml
name: Sync to GitHub Wiki

on:
  push:
    branches:
      - main
    paths:
      - "docs/**"
      - "README.md"
      - "AGENTS.md"

jobs:
  sync-wiki:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - name: Checkout repository
        uses: actions/checkout@v4

      - name: Sync documentation to wiki
        run: |
          git config --global user.name "github-actions[bot]"
          git config --global user.email "github-actions[bot]@users.noreply.github.com"

          # Clone the wiki repository
          git clone https://x-access-token:${{ secrets.GITHUB_TOKEN }}@github.com/${{ github.repository }}.wiki.git wiki

          # Copy documentation to wiki folder
          # rsync will merge the docs directory contents into the wiki root
          rsync -av --exclude='.*' docs/ wiki/
          cp README.md wiki/Home.md || true
          cp AGENTS.md wiki/AGENTS.md || true

          # Commit and push changes
          cd wiki
          git add .
          # Only commit and push if there are changes
          if ! git diff --cached --quiet; then
            git commit -m "docs: sync wiki from main repository [skip ci]"
            git push origin HEAD
          else
            echo "No changes to sync."
          fi
```

- [ ] **Step 2: Commit**

```bash
git add .github/workflows/wiki-sync.yml
git commit -m "ci: add github actions workflow to sync docs with wiki"
```
